# Support

Report extension bugs and feature requests in the
[`vscode-nomo` issue tracker](https://github.com/nomo-lang/vscode-nomo/issues).

Compiler and CLI issues belong in the
[`nomo` issue tracker](https://github.com/nomo-lang/nomo/issues). Language server
issues belong in the
[`nomo-lsp` issue tracker](https://github.com/nomo-lang/nomo-lsp/issues).
